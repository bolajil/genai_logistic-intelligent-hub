CLAUDE.md — Key Conventions

What CLAUDE.md will cover:
✦ Port assignments (9000–9009) — never use others without updating this file
✦ Startup commands: make dev (all services), make backend, make frontend
✦ Connector pattern — always implement base.py interface; never bypass via hardcoded data
✦ Agent pattern — agents never import connectors directly; always receive callbacks via api/agents.py
✦ Demo mode — simulator is always on in demo; never disable without updating .env
✦ Langfuse — every LLM call must carry a trace_id; never call llm.generate() without one
✦ Auth — all /api/* routes require JWT except /api/health and /api/auth/login
✦ Testing — run make test before any PR; Locust scenarios must pass at 50 users
✦ Cloud deploy — Terraform state in S3/GCS/Azure Blob; never apply manually
