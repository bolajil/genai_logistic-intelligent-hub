Page Descriptions

⬡ Dashboard
KPI cards, live sensor feed, agent activity log, shipment map, alert ticker. SSE-powered — updates without refresh.
⚡ Agents
Run any of the 4 agents manually. Select agent, fill input form or pick a demo preset, see structured result with LLM reasoning, SOP citations, and action plan.
🚚 Shipments
Searchable shipment table. Click any shipment for detail: temperature history chart, GPS trail, all events, compliance status, linked documents.
🌡 Alerts
All active and resolved alerts. Filter by severity, type, shipment. Each alert shows agent response, actions taken, SOP applied, and resolution time.
📊 Analytics
On-time %, temp compliance, food waste prevention, cost savings, agent run history. Charts powered by Recharts. Export to PDF/CSV.
📁 Documents
Ingest documents (drag-drop or URL). RAG query interface. Collection management. Search results with citations. SOP viewer.
⚙ Settings
Per-connection Demo/Real toggles for WMS, IoT, Docs. LLM provider selection. Embedding model. Notification channels (email, SMS, webhook).
🔒 Admin
User management, RBAC roles, API key management, audit log, Langfuse link, system health. Admin-role only.

6 Built-In Demo Scenarios

1
Temperature Breach Response — CHI-ATL-2025-089 dairy shipment breaches at 5.2°C. AnomalyResponder detects, queries SOPs, generates action plan, CustomerNotifier sends Whole Foods an email. Full agent chain in <10s.
2
Route Optimisation Under Risk — TX-CHI-2025-001 seafood at 68% spoilage risk. RouteAdvisor queries historical route performance, recommends Via Distribution Hub, calculates $920 cost vs $850 risk, shows savings.
3
Shift Handoff Report — OpsSummarizer generates 24h Chicago hub report: 4 events, 2 incidents, 94.2% on-time, LLM executive summary with recommendations for next shift. Exportable PDF.
4
SOP Natural Language Query — User asks "What do I do if seafood exceeds 4°C for 30 minutes?" RAG retrieves from lineage-sops.txt, LLM answers with citations, step-by-step procedure, escalation path.
5
Live IoT Simulator — Dashboard auto-demo mode: temperature gauges drift in real time, GPS markers move, automatic breach fires every ~5 min triggering the full AnomalyResponder → CustomerNotifier pipeline live.
6
Locust Load Test — One-click scenario: 50 concurrent users hitting all 4 agent endpoints + RAG query. Results in Locust UI (localhost:9009) + Langfuse shows per-agent LLM costs under load.
